WeRateDogs Twitter Data¶

Introduction:¶

The dataset that I have used for this project data wrangling (and analyzing and visualizing) is the tweet archive of Twitter user @dog_rates, also known as WeRateDogs. WeRateDogs is a Twitter account that rates people's dogs with a humorous comment about the dog.

Objective/goal of Project:¶

The aim of this project was to Perform data wrangling (gathering, assessing and cleaning) on provided three sources of data. Reporting on 1) data wrangling efforts and 2) data analyses and visualizations.

Gathering Data phase:¶

DataSets Used: Enhanced Twitter Archive This file ('twitter-archive-enhanced.csv')was downloaded manually.This contains basic tweet data for all 5000+ of their tweets, but not everything Image Predictions File: The tweet image predictions ('image-predictions.tsv'). This file was downloaded programmatically. Additional Data via the Twitter API: Using the tweet IDs in the WeRateDogs Twitter archive, we queried the Twitter API for each tweet's JSON data using Python's Tweepy library and stored each tweet's entire set of JSON data in a file called tweet_json.txt file. Accessing and Cleaning Data phase: On these three datasets we performed visual and programmatic assessments to have a clear understanding of data in these 3 datasets.
